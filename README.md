# Netease_down
> Version 1.0

可以通过链接下载网易云音乐歌单的所有音乐 

clone或者直接下载到本地

```
git clone https://github.com/wvdon/netease_down.git
```

安装需要的包

```
pip install -r requests.txt
```

运行下载

```
python netease.py
```

输入你要下载的歌单id :
